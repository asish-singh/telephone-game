# The Telephone Game

A traced, visual study of how an instruction changes as it travels from a human's intent, through an orchestrating AI (Claude), down to coding subagents (Codex), and finally into working code.

Every message between the layers is logged. Each layer's understanding of the task is captured and diffed against the original intent. The result is a set of figures, generated directly from the real trace, that show where meaning drifts, where it is recovered by feedback loops, and what that means for anyone who writes specifications for a living.
