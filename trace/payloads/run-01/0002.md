# Decomposition of the intent (Layer 1, Claude)

Product: a command line tool (Node.js, headless browser via Playwright) named "designdna".
Input: a URL. Output: an exhaustive DESIGN.md that an AI agent can use to replicate the site's look.

Feasibility call: CLI first, not a web or desktop app. The intent said "whatever feasible";
a CLI is the standard first form for a developer facing extraction tool and can be wrapped
in a web UI later.

Subtask A (codex-1), the extractor.
Render the URL in headless Chromium. Collect a structured JSON "design capture":
- color palette from computed styles (frequency ranked, foreground/background/border/accent roles)
- typography (families, sizes, weights, line heights, letter spacing, where each is used)
- spacing and layout system (container widths, grid/flex patterns, radii, shadows, breakpoints from media queries)
- every @keyframes animation and CSS transition, with the selectors and properties they animate,
  durations, easings, triggers where detectable (hover, scroll, load)
- backgrounds in detail, gradients, images, SVG, canvas or video hero elements, blend modes, filters
- inline SVGs and notable inline styles, captured verbatim when small
- section by section outline of the page (hero, nav, cards) with the key styles of each

Subtask B (codex-2), the writer.
Turn the design capture JSON into DESIGN.md, exhaustive and agent readable:
front section with theme summary, then tokens (colors, type, spacing, radii, shadows),
then layout system, then a section by section walkthrough, then an "Animations" chapter
reproducing every keyframe and transition as code blocks with plain descriptions,
then "Backgrounds and effects", then a replication checklist. Deterministic, no AI calls.

Interface contract between A and B: capture.json schema defined by codex-1 and written to
docs/capture-schema.md; codex-2 codes against that file.

Acceptance test: `node bin/designdna.js https://www.shopify.com/editions/winter2026 -o DESIGN.md`
completes and the DESIGN.md contains real keyframes and gradient definitions from that page.
